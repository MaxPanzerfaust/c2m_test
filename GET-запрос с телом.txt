Вопрос: Можно ли послать GET запрос с телом. 
Технически это вполне возможно,но так делать не принято.  В соответствии со спецификациями HTTP 1.1 рекомендовано на стороне сервера игнорировать тело GET-запроса ,если только нет весомых причин для обратного,как например большое количество передаваемых параметров. Postman без проблем отправляет get запросы с телом, а Spring MVC позволяет их обрабатывать. Однако,например, некоторые  java-библиотеки, предоставляющие http-клиент(httpOK), могут попросту не дать вам отправить такой запрос, а swagger при автогенерации документации вашего api проигнорирует тело запроса,которое вы ожидаете
